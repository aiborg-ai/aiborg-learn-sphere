# Bundle Size Monitoring - Files Created

This document lists all files created for the bundle size monitoring system.

## Scripts

### `/home/vik/aiborg_CC/aiborg-learn-sphere/scripts/`

1. **monitor-bundle-size.ts** (Already existed, now enhanced)
   - Core monitoring script
   - Validates bundle sizes against thresholds
   - Generates `bundle-size-report.json`
   - Color-coded terminal output

2. **track-bundle-history.ts** (NEW)
   - Tracks bundle size history (last 50 builds)
   - Records git commit and branch info
   - Calculates trends and statistics
   - Generates `bundle-size-history.json`

3. **compare-bundle-sizes.ts** (NEW)
   - Compares current vs previous builds
   - Identifies new/removed bundles
   - Highlights significant changes
   - Percentage change calculations

4. **analyze-bundle.sh** (Already existed)
   - Quick terminal-based analysis
   - Shows largest bundles
   - Identifies chunks over threshold
